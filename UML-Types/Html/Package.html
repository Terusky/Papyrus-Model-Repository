説明

パッケージには、適用されているプロファイルを示す1つ以上のプロファイルアプリケーションを含めることができます。

プロファイルはパッケージであるため、パッケージだけでなくプロファイルにも適用できます。

PackageはTemplateableElementを特殊化し、PackageableElementはParameterableElementを特殊化して、パッケージをテンプレートとして、PackageableElementをテンプレートパラメーターとして使用できることを指定します。

パッケージは、要素をグループ化するために使用され、グループ化された要素の名前空間を提供します。
