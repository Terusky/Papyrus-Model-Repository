説明

パッケージには、適用されているプロファイルを示す1つ以上のプロファイルアプリケーションを含めることができます。

プロファイルはパッケージであるため、パッケージだけでなくプロファイルにも適用できます。

PackageはTemplateableElementを特殊化し、PackageableElementはParameterableElementを特殊化して、パッケージをテンプレートとして、PackageableElementをテンプレートパラメーターとして使用できることを指定します。

パッケージは、要素をグループ化するために使用され、グループ化された要素の名前空間を提供します。

プロパティ 説明
URI 多くの目的に使用できるパッケージの識別子を提供します。 URIは、IETF URI仕様（RFC 2396 http://www.ietf.org/rfc/rfc2396.txt）に準拠した、パッケージの普遍的に一意のIDであり、これらの構文規則に準拠している必要があります。
nestedPackage Packagesであるパッケージ化された要素を参照します。
nestingPackage このパッケージを所有するパッケージを参照します。
ownedStereotype パッケージが所有するステレオタイプを参照します。
ownedType タイプであるパッケージ化された要素を参照します。
packageMerge このパッケージが所有するPackageMergesを参照します。
packagedElement このパッケージが所有するパッケージ化可能な要素を指定します。
profileApplication パッケージに適用されているプロファイルを示すProfileApplicationsを参照します。
